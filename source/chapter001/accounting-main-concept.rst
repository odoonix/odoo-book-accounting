

معادله اساسی حسابداری
=========================================================================================

پایه حسابداری دوطرفه، معادله زیر است:

.. math::

    دارایی‌ها = بدهی‌ها + سرمایه

این معادله همیشه برقرار است.

مثال ساده
-------------

فرض کنید:

- 2,000,000 ریال سرمایه شخصی وارد مؤسسه می‌کنید.
- 600,000 ریال وام می‌گیرید.

پس داریم:

.. code-block:: none

    دارایی‌ها = 2,600,000
    بدهی‌ها = 600,000
    سرمایه = 2,000,000

و معادله برقرار است:

.. math::

    2,600,000 = 600,000 + 2,000,000
